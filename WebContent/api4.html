API

API

Vårt API returnerar svar i JSON-format. Vi använder endast GET-anrop för att hämta data i form av information om filmen från OMDB och trailer från TrailerAddict. Det finns även en mindre avancerad funktion som visar aktuell API-version.

Exempel

Exempel 1:

Låt oss anta att användaren vill hämta information och trailer för filmen ”Heat”.

Exempel 2:

Låt oss anta att användaren vill hämta information och trailer för filmen ”The Shining”.

Exempel 3:

Låt oss anta att användaren vill hämta information och trailer för en film som inte existerar. I det här fallet anropas filmtiteln ”Turn to page 394”.

Hämta information om en film

Ändpunkt: 

api/collection/{movietitle}

Exempel 1: api/collection/heat

Exempel 2: api/collection/the-shining

Exempel 3: api/collection/turn-to-page-394

Metod:

GET. Hämta information om en film.

Parameter:

Filmtitel.

Exempel 1: heat

Exempel 2: the-shining

Exempel 3: turn-to-page-394

Fullständig länk:

http://localhost:8080/MoviezClient/api/collection/{movietitle}

Hämta trailer

Ändpunkt: 

api/watchtrailer/{movietitle}

Exempel 1: api/watchtrailer/heat

Exempel 2: api/watchtrailer/the-shining

Exempel 3: api/watchtrailer/turn-to-page-394

Parameter:

Filmtitel.

Hämta information om aktuell API-version

Ändpunkt: 

api/version/check

Metod:

GET. Hämta information om aktuell API-version.

Parameter:

check

Fullständig länk:

http://localhost:8080/MoviezClient/api/version/check

Returvärde:

Exempelsvar:

v 1.0
